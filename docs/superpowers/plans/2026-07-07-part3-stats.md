# Part 3 — Statistics API Implementation Plan

> ⚠️ **SUPERSEDED by the v2 architecture.** Statistics now live in the
> **analytics** service (read side over the Mongo replica) — see
> `2026-07-07-service-analytics.md`. Kept only as an aggregation-logic reference.

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement `GET /v1/stats/summary?configId=&from=&to=` returning aggregated statistics (totals, per-category, per-action, top attackers, top targeted paths) for a configuration and time range.

**Architecture:** Aggregation is an outbound-port capability: extend `EventStore` with `summarize(StatisticsQuery)`, implemented in the in-memory adapter now (Mongo/Postgres adapters in later plans). The `SummarizeStatistics` application service delegates to the port; `StatsController` maps the domain `StatisticsSummary` to the assignment's exact JSON via a response DTO. Domain/application stay pure; only the controller knows JSON.

**Tech Stack:** Java 21, Spring Boot 3.3.5 (web, test), Maven, JUnit 5 + AssertJ + MockMvc.

## Global Constraints

- **Java 21**, base package `com.akamai.wsa`, records, immutability, no input mutation.
- `domain` imports no Spring/Jackson/storage types.
- **Aggregation runs against the `EventStore` port**, never by pulling all rows into the controller.
- **Response shape is the assignment's literal JSON** — no `{success,data}` envelope.
- `configId` optional (null → all configs); `from`/`to` optional (absent → `TimeRange.unbounded()`).
- `topAttackers` / `topTargetedPaths` = top **10** by count (`StatisticsSummary.TOP_LIMIT`); ties broken deterministically (by IP string / path string ascending).
- Averages rendered to **one decimal place** in JSON.
- Conventional commits; local dev port **8081**; milestone tag `v0.3-stats`.
- Reuse existing records in `com.akamai.wsa.application.stats` (`StatisticsQuery`, `StatisticsSummary`, `CategoryStatistics`, `AttackerStatistics`, `PathStatistics`) and `com.akamai.wsa.domain.model.TimeRange`.

Add to `InMemoryEventStore` (imports: `java.util.*`, `java.util.stream.Collectors`, the stats records, `Action`, `AttackCategory`, `SecurityEvent`):
```java
    @Override
    public StatisticsSummary summarize(StatisticsQuery statisticsQuery) {
        List<SecurityEvent> matching = events.stream()
                .filter(securityEvent -> statisticsQuery.configId() == null
                        || securityEvent.configId() == statisticsQuery.configId())
                .filter(securityEvent -> statisticsQuery.timeRange().includes(securityEvent.timestamp()))
                .toList();

        Map<AttackCategory, CategoryStatistics> byCategory = matching.stream()
                .collect(Collectors.groupingBy(SecurityEvent::category,
                        Collectors.collectingAndThen(Collectors.toList(),
                                grouped -> new CategoryStatistics(grouped.size(), averageThreatScore(grouped)))));

        Map<Action, Long> byAction = matching.stream()
                .collect(Collectors.groupingBy(SecurityEvent::action, Collectors.counting()));

        List<AttackerStatistics> topAttackers = matching.stream()
                .collect(Collectors.groupingBy(SecurityEvent::clientIp, Collectors.toList()))
                .entrySet().stream()
                .map(entry -> new AttackerStatistics(entry.getKey(), entry.getValue().size(),
                        averageThreatScore(entry.getValue())))
                .sorted(Comparator.comparingLong(AttackerStatistics::count).reversed()
                        .thenComparing(attacker -> attacker.clientIp().value()))
                .limit(StatisticsSummary.TOP_LIMIT)
                .toList();

        List<PathStatistics> topTargetedPaths = matching.stream()
                .collect(Collectors.groupingBy(SecurityEvent::path, Collectors.counting()))
                .entrySet().stream()
                .map(entry -> new PathStatistics(entry.getKey(), entry.getValue()))
                .sorted(Comparator.comparingLong(PathStatistics::count).reversed()
                        .thenComparing(PathStatistics::path))
                .limit(StatisticsSummary.TOP_LIMIT)
                .toList();

        return new StatisticsSummary(statisticsQuery.configId(), statisticsQuery.timeRange(),
                matching.size(), byCategory, byAction, topAttackers, topTargetedPaths);
    }

    private static double averageThreatScore(List<SecurityEvent> group) {
        return group.stream()
                .mapToInt(securityEvent -> securityEvent.threatScore().value())
                .average()
                .orElse(0.0);
    }
```

package com.akamai.wsa.interfaces.rest.dto;

import com.akamai.wsa.application.stats.StatisticsSummary;
import com.akamai.wsa.domain.model.Action;
import com.akamai.wsa.domain.model.AttackCategory;

import java.time.Instant;
import java.util.List;
import java.util.Map;
import java.util.stream.Collectors;

/**
 * Wire representation of a {@link StatisticsSummary} in the assignment's exact shape.
 */
public record StatisticsSummaryResponse(
        Integer configId,
        TimeRangeView timeRange,
        long totalEvents,
        Map<String, CategoryView> byCategory,
        Map<String, Long> byAction,
        List<AttackerView> topAttackers,
        List<PathView> topTargetedPaths
) {
    public record TimeRangeView(Instant from, Instant to) {
    }

    public record CategoryView(long count, double avgThreatScore) {
    }

    public record AttackerView(String clientIp, long count, double avgThreatScore) {
    }

    public record PathView(String path, long count) {
    }

    public static StatisticsSummaryResponse from(StatisticsSummary summary) {
        Map<String, CategoryView> byCategory = summary.byCategory().entrySet().stream()
                .collect(Collectors.toMap(
                        entry -> entry.getKey().name(),
                        entry -> new CategoryView(entry.getValue().count(),
                                roundToOneDecimal(entry.getValue().averageThreatScore()))));

        Map<String, Long> byAction = summary.byAction().entrySet().stream()
                .collect(Collectors.toMap(entry -> entry.getKey().name(), Map.Entry::getValue));

        List<AttackerView> topAttackers = summary.topAttackers().stream()
                .map(attacker -> new AttackerView(attacker.clientIp().value(), attacker.count(),
                        roundToOneDecimal(attacker.averageThreatScore())))
                .toList();

        List<PathView> topTargetedPaths = summary.topTargetedPaths().stream()
                .map(path -> new PathView(path.path(), path.count()))
                .toList();

        return new StatisticsSummaryResponse(
                summary.configId(),
                new TimeRangeView(summary.timeRange().from(), summary.timeRange().to()),
                summary.totalEvents(),
                byCategory,
                byAction,
                topAttackers,
                topTargetedPaths);
    }

    private static double roundToOneDecimal(double value) {
        return Math.round(value * 10.0) / 10.0;
    }

    private static Action unusedActionAnchor() {
        return null; // kept only so the Action import documents the source enum; safe to delete
    }
}
```

package com.akamai.wsa.interfaces.rest;

import com.akamai.wsa.application.stats.StatisticsQuery;
import com.akamai.wsa.application.stats.StatisticsSummary;
import com.akamai.wsa.application.stats.SummarizeStatistics;
import com.akamai.wsa.domain.model.TimeRange;
import com.akamai.wsa.interfaces.rest.dto.StatisticsSummaryResponse;
import org.springframework.format.annotation.DateTimeFormat;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

import java.time.Instant;

@RestController
@RequestMapping("/v1/stats")
class StatsController {

    private final SummarizeStatistics summarizeStatistics;

    StatsController(SummarizeStatistics summarizeStatistics) {
        this.summarizeStatistics = summarizeStatistics;
    }

    @GetMapping("/summary")
    StatisticsSummaryResponse summary(
            @RequestParam(required = false) Integer configId,
            @RequestParam(required = false) @DateTimeFormat(iso = DateTimeFormat.ISO.DATE_TIME) Instant from,
            @RequestParam(required = false) @DateTimeFormat(iso = DateTimeFormat.ISO.DATE_TIME) Instant to) {

        StatisticsQuery query = new StatisticsQuery(configId, new TimeRange(from, to));
        StatisticsSummary summary = summarizeStatistics.summarize(query);
        return StatisticsSummaryResponse.from(summary);
    }
}
```
